# YouSync

유싱크는 실시간 분석서버와 연동된 온라인 더빙 웹 프로젝트입니다.  
실시간으로 분석하며 sync를 맞춰보세요!

## 주요기능

### 1. **더빙 체험**  
   내가 원하는 영상을 클릭해 더빙 체험을 진행하세요
